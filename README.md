me-shumei-open-oks-xiaomibbs
============================
Android《一键签到》的小米手机BBS签到插件

本项目采用 GPL 授权协议，欢迎大家在这个基础上进行改进，并与大家分享。

《一键签到》官方网站：<http://oks.shumei.me>
